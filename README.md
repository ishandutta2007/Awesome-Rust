## Projects & Applications

### CLI Tools
- [ripgrep (rg)](https://github.com/BurntSushi/ripgrep) ‚Äì blazingly fast grep
- [bat](https://github.com/sharkdp/bat) ‚Äì `cat` with syntax highlighting
- [fd](https://github.com/sharkdp/fd) ‚Äì user-friendly `find`
- [zoxide](https://github.com/ajeetdsouza/zoxide) ‚Äì smarter `cd` with frecency
- [eza](https://github.com/eza-community/eza) ‚Äì modern replacement for `ls`

### Web & Full-stack
- [Zola](https://github.com/getzola/zola) ‚Äì fast static site generator
- [Leptos](https://github.com/leptos-rs/leptos) ‚Äì fine-grained reactive web framework (full-stack)
- [Tauri](https://github.com/tauri-apps/tauri) ‚Äì build lightweight desktop apps with web frontend
- [Axum](https://github.com/tokio-rs/axum) + [Tower](https://github.com/tower-rs/tower) ‚Äì modular web framework (used in production by many)

### Games & Graphics
- [Bevy](https://github.com/bevyengine/bevy) ‚Äì data-driven game engine
- [Veloren](https://github.com/Veloren/Veloren) ‚Äì open-world multiplayer voxel RPG
- [Macroquad](https://github.com/not-fl3/macroquad) ‚Äì simple & fast 2D/3D game framework

### System & Embedded
- [Redox OS](https://github.com/redox-os/redox) ‚Äì full Rust operating system
- [Tock OS](https://github.com/tock/tock) ‚Äì embedded OS for microcontrollers
- [esp-rs](https://github.com/esp-rs) ‚Äì Rust on ESP32/ESP8266

### Other Cool Apps
- [Alacritty](https://github.com/alacritty/alacritty) ‚Äì GPU-accelerated terminal emulator
- [Helix](https://github.com/helix-editor/helix) ‚Äì modal text editor (like Kakoune + Neovim)
- [Lapce](https://github.com/lapce/lapce) ‚Äì lightning-fast code editor written in Rust
